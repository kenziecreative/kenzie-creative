Run the **daily** writing ritual.

Use the `goal-setting-pulse` skill for the **daily** cadence and follow its steps exactly.
It asks where the user is writing today (a different location than yesterday — the
adaptation-defeating protocol), prompts them to write their active Objective(s) by hand and
confirm, and appends a short dated entry to `goals/journal.md`.

This is meant to be terse — 60 to 90 seconds. Don't turn it into journaling.
