Run the **Systems** stage (Setup Stage 5) of the goal-setting method.

Use the `goal-setting-setup-stage` skill for the **systems** stage and follow its steps
exactly. It reads `${CLAUDE_PLUGIN_ROOT}/reference/stages/05-systems.md`, designs one weekly
system per Objective against the Four Laws of Behavior Change, and writes each system to
`goals/active.md` under its Objective.

**Hard constraints:** every system must have a real trigger (time, location, or habit-stack)
— the skill rejects trigger-less systems as "hopes, not systems" — and only one system per
active anchor area at initial setup.
