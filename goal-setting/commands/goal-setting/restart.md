Run the **Restart Protocol**.

Use the `goal-setting-restart` skill and follow its steps exactly. It walks the five-step
recovery from playbook Appendix B: skip the guilt, re-score the Anchor Areas, restart with one
system (not all of them), resume the daily writing tomorrow, and defer the next quarterly if
it's close. Writes a restart entry to `goals/journal.md` and sets the relevant flags in
`goals/STATE.md`.

**Hard constraint:** the restart brings back exactly one system. The skill auto-pauses the
others — relaunching at full intensity is how the restart fails.

Falling off is data, not a verdict. The protocol exists so a lapse interrupts the practice
rather than ending it.
