# Reference Guides

These guides help readers use the pattern library in live situations, where the hard part is often not "what is the definition?" but "which pattern is this, what else could it be, and what should I do next?"

Use them as routing tools, not verdict machines. A good label should increase clarity, options, and accountability. If a label makes you more certain but less curious, slow down.

## Guides

- [High-Risk Mislabels](./high-risk-mislabels.md) - Common places where a useful concept can become an over-applied accusation.
- [Debate and Information Overload](./debate-and-information-overload.md) - Distinguishing overload, framing, authority, and persistence problems.
- [User Journey Guide](./user-journey-guide.md) - Five practical routes through the corpus: name what is happening, check your own thinking, respond to manipulation, make a decision, and practice pattern recognition.
