# Begging The Question

> A logical fallacy in which the premise of an argument presupposes the truth of its conclusion; in other words, the argument assumes what it is trying to prove.

## Description

Begging the question is a form of circular reasoning. The conclusion is smuggled into the premise, so the argument appears to prove something while actually assuming it from the start.

The phrase is often misused to mean "raises the question." In the fallacy sense, it means the argument depends on what it is supposed to establish.

This is especially easy to miss when the premise and conclusion use different words with similar meanings. "This policy is necessary because it is essential" sounds like reasoning, but it mostly repeats the claim.

## Flawed Move

The flawed move is offering a reason that only works if the conclusion is already accepted.

Common versions include:

- "This source is trustworthy because it tells the truth."
- "The plan will succeed because it is the winning strategy."
- "They are guilty because only a guilty person would act that way."
- "Our product is the best because it is superior."

## Why It Persuades

Circular arguments can feel satisfying because they use confident language and familiar labels. If the conclusion already feels plausible, the repeated premise may pass as proof.

They also hide inside shared assumptions. In groups with strong norms, an argument may not need evidence because everyone already treats the conclusion as obvious.

## What It Avoids

Begging the question avoids independent support:

- What evidence would persuade someone who does not already agree?
- Which premise can be tested separately?
- Are the key terms doing real work or merely restating each other?
- What would count against the conclusion?

## Common False Positives

Not every short argument is circular. Some claims are shorthand for a longer body of evidence already established in context.

The test is whether the reason can stand apart from the conclusion. If the speaker can unpack the premise into independent evidence, the argument may simply have been compressed.

## How to Respond

Ask for a reason that does not depend on accepting the conclusion.

- "What independent evidence supports that?"
- "How would you explain this to someone who does not already agree?"
- "Can we define the key terms without using the conclusion?"
- "What would make this claim false?"

If you are making the argument, replace repeated labels with observable evidence, mechanisms, comparisons, or consequences.

## Decision Prompt

> Does this argument prove the conclusion, or does it only restate it in different words?
