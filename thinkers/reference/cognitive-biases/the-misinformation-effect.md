# The Misinformation Effect

> The tendency for post-event information to interfere with the memory of the original event.

## Description

The misinformation effect is the tendency for information encountered after an event to alter, contaminate, or interfere with memory of the original event.

Memory is not a recording that gets replayed unchanged. It is reconstructed, and later questions, stories, images, labels, or social pressure can become woven into the remembered event.

## How It Works

When people retrieve a memory, they rebuild it from fragments, expectations, and context. New information can supply details that feel familiar later.

Leading questions are especially powerful because they smuggle assumptions into the reconstruction process. "How fast was the car going when it smashed into the other car?" can shape memory differently than "How fast was the car going when it contacted the other car?"

## Recognition Cues

- Details become more vivid after hearing someone else's version.
- A question contains assumptions that were not in the original observation.
- Multiple retellings slowly converge on a shared but unverified story.
- Confidence increases even though the memory source is unclear.
- Later media, photos, or commentary feel like original memory.

## Signature Phrasing

- "memories are reconstructions"
- "altered by new information"
- "leading question"
- "game of telephone"
- "Now that you mention it, I remember..."

## Ask Yourself

> How can I verify this memory against independent, contemporaneous evidence?

## Common False Positives

Updating a memory with real evidence is not automatically distortion. New documents, recordings, or multiple independent accounts can correct an inaccurate memory.

The bias appears when later information is absorbed as if it were originally perceived, especially when the source of the detail is forgotten.

## What Helps

- Record important observations as soon as possible.
- Use neutral, open-ended questions before specific ones.
- Separate "I saw" from "I later heard."
- Preserve source trails: notes, timestamps, documents, messages.
- In group reviews, gather independent accounts before discussion.

## Evidence Status

Evidence status: well-established in memory research, especially around eyewitness memory, suggestion, and post-event information.
