# The Categorical Imperative

> An ethical test that asks whether the rule behind your action could be followed by everyone without contradiction or disrespect.

## Description

The categorical imperative is most useful as a self-exception detector. It asks whether you are giving yourself permission to do something that would break trust, dignity, or social order if everyone did it under similar conditions.

Rather than judging only the outcome, it examines the principle underneath the action. "Can I lie when it benefits me?" is not just about this one lie. It is about whether communication can survive if that rule becomes normal.

## Core Test

What rule am I acting on, and could I honestly will that everyone follow that same rule in comparable circumstances?

A second version asks: am I treating this person as an end with their own agency, or merely as a tool for my goal?

## What It Helps You See

This lens reveals rationalizations. It catches special pleading, deception dressed as necessity, pressure tactics, and convenient exceptions to standards you expect from others.

It also protects dignity. Some actions may produce good outcomes while still treating people as disposable instruments. The categorical imperative forces that issue into view.

## When to Use

Use it when you are tempted to lie, conceal, manipulate, exploit a loophole, break a promise, or justify behavior by saying "just this once."

It is especially useful for leaders and institutions because repeated exceptions from powerful actors become norms for everyone else.

## When It Backfires

It can become rigid when duties conflict. Telling the truth, protecting someone, honoring confidentiality, and preventing harm can pull in different directions.

It also needs careful formulation. A rule can be described so narrowly that it excuses anything, or so broadly that it forbids ordinary judgment.

## Practical Use

- Before using a deceptive marketing claim, ask what would happen if every seller used the same practice.
- Before making an exception for yourself, ask whether you would accept the same exception from someone with less status.
- Before pressuring someone, ask whether the tactic respects their ability to choose.

## Decision Prompt

What rule would my action teach, and could I live in a world where that rule became normal?
