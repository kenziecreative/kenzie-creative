# The Decoy Effect

> A choice-design effect where an inferior third option makes a target option look more attractive.

## Description

The decoy effect appears when a choice set includes an option that is not meant to be chosen. Its purpose is to change the comparison between the real options.

This belongs in strategy because choice architecture shapes decisions. It also belongs near persuasion ethics because the same mechanism can clarify tradeoffs or steer people without their awareness.

## Mechanism

People evaluate options comparatively. A decoy is asymmetrically dominated: it is clearly worse than the target option on relevant dimensions, but not clearly worse than every alternative. Its presence makes the target look like the obvious better value.

The decision changes not because the target improved, but because the comparison set changed.

## Strategic Use

Use this lens when designing or evaluating menus, plans, offers, pricing, packages, hiring options, product tiers, or tradeoff sets.

Ethical choice design can help people compare real tradeoffs. Manipulative choice design hides the fact that one option is present mainly to steer.

## When It Backfires

It backfires when the decoy is obvious, insulting, or irrelevant. People do not like feeling managed.

It also backfires when teams optimize for conversion while damaging trust, retention, or fit.

## Ethical Boundary

Ethical use makes meaningful differences easier to see. Manipulative use creates fake options to produce a choice the person might not make under clearer comparison.

The ethical question is whether the choice set improves judgment or quietly removes it.

## Practical Examples

- A software company offers tiers that clearly reflect different customer needs rather than a fake middle option.
- A buyer notices that one package seems designed only to make the premium package look reasonable.
- A facilitator removes irrelevant options from a decision so the group compares real tradeoffs.

## Decision Prompt

Is this option here because someone might choose it, or because it changes how another option feels?
