# Whataboutism

> Deflecting accountability by pointing to another wrong instead of addressing the issue at hand.

## Description

Whataboutism responds to criticism with a different criticism: "What about them?" The new issue may be real, but it functions as a diversion from the original claim.

The move is powerful because it appeals to fairness. If others are guilty too, the target hopes the current accountability will feel selective or hypocritical.

## Mechanism

Whataboutism shifts the burden and changes the topic. The conversation moves from "Did this happen?" to "Are the critics pure enough to say so?"

It often creates false equivalence, implying that because another wrong exists, the present wrong does not need to be addressed.

## What It Avoids

- The original claim.
- Accountability for the specific actor or action.
- Proportional comparison.
- The possibility that multiple wrongs can be true.
- Repair or resolution.

## Recognition Cues

- The response begins with "What about..."
- The new issue is introduced before the first is answered.
- Hypocrisy is used to avoid factual evaluation.
- The speaker treats another wrong as cancellation rather than context.

## Common False Positives

Comparisons can be legitimate when they test consistency, standards, or proportionality.

The question is whether the comparison clarifies the current issue or replaces it.

## How to Respond

Acknowledge the other issue if it is real, then return to the current one: "That may matter too. Right now, we are discussing this claim."

If needed, create separate lanes: first this accountability, then the comparison.

## Decision Prompt

Does this comparison help evaluate the current issue, or is it being used to escape it?
