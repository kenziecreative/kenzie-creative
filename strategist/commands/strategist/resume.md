Resume the Strategist project in this directory.

Use the `strategist-resume` skill and follow its steps exactly. It rebuilds the working
state from `strategy/STATE.md` and `strategy/brief.md` — stance, calibration,
hypotheses, mid-stage position — executes any private prep quietly, briefs the user on
where things stand, and waits for their confirmation before continuing. The files win
over conversation memory, silently.
