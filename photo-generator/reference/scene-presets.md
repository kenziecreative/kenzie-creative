# Scene Presets Reference

This file provides quick-start combinations for common photography scenarios. Use these as starting points and customize as needed.

---

## How to Use Presets

Each preset provides a complete stack:
1. **Camera Body** - The sensor and color science
2. **Lens** - Focal length and aperture
3. **Lighting** - The light setup
4. **Post-Processing** - The final color grade

You can use these directly or swap individual elements based on your specific needs.

## Preset Modification Tips

**Want more drama?**
- Switch to Rembrandt or Low-Key lighting
- Use Moody Film or Cinematic post-processing
- Increase contrast in post description

**Want softer/romantic?**
- Use Golden Hour or Window Light
- Choose Matte Film or Portra post-processing
- Describe "soft," "lifted shadows"

**Want more modern/clean?**
- Use High-Key lighting
- Choose Clean Commercial or High-Key Clean post
- Describe "minimal," "crisp," "pure"

**Want vintage feel?**
- Use any natural lighting
- Choose film stocks (Portra, Ektar, etc.)
- Add "film grain," "lifted blacks," "faded"
